# CLAUDE.md Template — Next.js 15 + SQLite SaaS

An opinionated, production-ready `CLAUDE.md` for Next.js 15 projects with SQLite.

## Usage

1. Create a new Next.js 15 project
2. Copy `CLAUDE.md` to the project root
3. Claude Code will automatically read it

## What It Covers

- Project structure conventions
- Database schema patterns with Drizzle ORM
- Component architecture (server-first)
- Anti-patterns to avoid
- Authentication with NextAuth v5
- Styling with Tailwind CSS + shadcn/ui
