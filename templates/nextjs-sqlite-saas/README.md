# Next.js 15 + SQLite SaaS CLAUDE.md Template

An opinionated, production-ready `CLAUDE.md` for SaaS projects built with Next.js 15 App Router and SQLite.

## Usage

Copy `CLAUDE.md` to the root of your Next.js 15 + SQLite project.

## What it covers

- Project structure and naming conventions
- Database migration rules and patterns
- Dev workflow commands
- Security best practices
- Anti-patterns to avoid

## Requirements

- Next.js 15 App Router
- SQLite (better-sqlite3 or Turso)
- Drizzle ORM
- TypeScript strict mode
